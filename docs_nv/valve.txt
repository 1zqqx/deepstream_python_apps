Element Properties:

  drop                : Whether to drop buffers and events or let them through
                        flags: 可读, 可写, 可以在NULL、READY、PAUSED或PLAYING状态下改变
                        Boolean. Default: false
  
  drop-mode           : The drop mode to use
                        flags: 可读, 可写, 只能在NULL、或READY状态下改变
                        Enum "GstValveDropMode" Default: 0, "drop-all"
                           (0): drop-all         - Drop all buffers and events
                           (1): forward-sticky-events - Drop all buffers but forward sticky events
                           (2): transform-to-gap - Convert all dropped buffers into gap events and forward sticky events
  
  name                : The name of the object
                        flags: 可读, 可写
                        String. Default: "valve0"
